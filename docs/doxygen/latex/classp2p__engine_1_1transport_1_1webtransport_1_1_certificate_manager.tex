\begin{DoxyVerb}Manages self-signed certificates for WebTransport

Per libp2p spec:
- Certificates are self-signed
- Validity period ≤ 14 days
- Cannot use RSA keys
- Certificate hashes are advertised in multiaddr
\end{DoxyVerb}
