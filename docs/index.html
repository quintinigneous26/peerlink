What's Inside

A Complete P2P Protocol Stack

Everything you need to build production P2P applications, implemented in modern C++20.

Security

TLS 1.3 + Noise

TLS 1.3 with 1-RTT handshake and hardware acceleration (AES-NI). Noise Protocol for cert-free IoT scenarios.

Transport

TCP · QUIC · WebRTC · WebTransport

Four transport layers: TCP for reliability, QUIC for low latency, WebRTC for browser-to-browser, WebTransport for HTTP/3.

Protocol

mplex + yamux Stream Multiplexing

Two stream multiplexing protocols. mplex for lightweight clients, yamux for high-throughput nodes with backpressure control.

Routing

Kademlia DHT

Distributed hash table for decentralized peer discovery. XOR distance metric, K-bucket routing, concurrent lookup optimization.

PubSub

GossipSub v1.1

Scalable publish/subscribe with topic mesh control, peer scoring, and adaptive gossip. Production-tested in libp2p networks.

NAT Traversal

STUN · TURN · ICE · Hole Punching

Four-layer NAT traversal: STUN detection, ICE candidate gathering, UDP hole punching, TURN TCP relay fallback.

NAT Traversal

Four Layers. 80%+ Success Rate.

The hardest part of P2P is NAT traversal. PeerLink implements a four-layer approach that works in the real world — even on symmetric NAT combos.

1

STUN

Detect NAT type and public IP/port mapping behavior

2

ICE

Gather all candidate addresses from all interfaces

3

Hole Punching

UDP simultaneous open — direct connection attempt

4

TURN Relay

TCP relay fallback when direct connection fails

No port forwarding required on either end.

Get Started

Build Your First P2P App

CMake 3.20+, C++20, OpenSSL 3.0. No other dependencies.

// p2p-cpp/examples/connect.cpp #include <p2p/engine.hpp> int main() { p2p::Config config; config.listen_addresses = {"/ip4/0.0.0.0/tcp/0"}; config.enable_webrtc = true; auto engine = p2p::Engine::Create(config); engine->Start(); auto conn = engine->Connect( p2p::PeerId::FromString("QmPeer...") ); conn->Send("Hello, P2P!"); engine->Stop(); return 0; }
$ cmake -B build -DCMAKE_BUILD_TYPE=Release
$ cmake --build build -j$(nproc)
$ cd build && ctest -V
