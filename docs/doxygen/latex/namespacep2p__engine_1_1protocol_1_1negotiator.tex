\doxysubsection{详细描述}
\begin{DoxyVerb}multistream-select 协议协商器

实现 libp2p 兼容的协议协商流程:
1. 双方交换 multistream 协议 ID
2. 发起方提议协议
3. 接收方响应支持 (echo) 或不支持 (na)

参考: https://github.com/libp2p/specs/blob/master/connections/README.md
