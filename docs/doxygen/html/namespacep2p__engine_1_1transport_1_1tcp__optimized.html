详细描述

优化的 TCP 传输实现

提供高性能 TCP 传输，支持：
- TCP Fast Open (TFO)
- 连接池
- Nagle 算法禁用
- Keep-Alive 优化

目标：连接延迟 < 20ms (对标 go-libp2p)
