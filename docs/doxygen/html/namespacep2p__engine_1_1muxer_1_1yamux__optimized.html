详细描述

优化的 Yamux 流复用实现

对标 go-libp2p yamux 性能：
- 流创建延迟: < 2ms
- 消息吞吐: > 200 Mbps
- 并发流: > 1000

优化：
1. 流 ID 预分配池
2. 批量流创建
3. 零拷贝帧处理
4. 读写锁分离
