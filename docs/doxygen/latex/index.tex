\hypertarget{index_intro}{}\doxysection{\texorpdfstring{项目介绍}{项目介绍}}\label{index_intro}
P2P Platform 是一个基于\+Web\+RTC技术的去中心化\+P2\+P通信平台，提供完整的\+P2\+P连接解决方案。 平台包含\+STUN服务器、\+TURN中继服务器、信令服务器、\+DID身份服务和客户端\+SDK等核心组件。\hypertarget{index_features}{}\doxysection{\texorpdfstring{主要特性}{主要特性}}\label{index_features}

\begin{DoxyItemize}
\item {\bfseries{智能\+NAT穿透}}:\+ 支持多种\+NAT类型的穿透，包括对称型\+NAT的端口预测
\item {\bfseries{运营商差异化}}:\+ 针对不同运营商的网络特性进行优化
\item {\bfseries{自动降级}}:\+ 连接失败时自动降级到中继服务
\item {\bfseries{心跳保活}}:\+ 保持连接活跃，防止\+NAT映射超时
\item {\bfseries{事件驱动}}:\+ 基于事件总线的异步架构
\item {\bfseries{完整的\+SDK}}:\+ 提供易用的客户端\+SDK
\end{DoxyItemize}\hypertarget{index_architecture}{}\doxysection{\texorpdfstring{架构设计}{架构设计}}\label{index_architecture}

如果\+NAT穿透失败，请检查：
\begin{DoxyItemize}
\item STUN服务器是否可达
\item 防火墙是否阻止了\+UDP/\+\+TCP连接
\item 对端是否在线
\item 网络延迟是否过高
\end{DoxyItemize}\hypertarget{index_connection_issues}{}\doxysubsection{\texorpdfstring{连接问题}{连接问题}}\label{index_connection_issues}
如果连接建立失败，请检查：
\begin{DoxyItemize}
\item 信令服务器是否正常运行
\item 对端信息是否正确
\item 网络环境是否支持\+P2\+P连接
\end{DoxyItemize}\hypertarget{index_performance}{}\doxysection{\texorpdfstring{性能优化}{性能优化}}\label{index_performance}
