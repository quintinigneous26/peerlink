## з»„д»¶иҜҙжҳҺ

### 1. STUNжңҚеҠЎеҷЁ
- **еҚҸи®®**: RFC 5389
- **еҠҹиғҪ**:
  - иҺ·еҸ–е®ўжҲ·з«Ҝе…¬зҪ‘IP:Port
  - жЈҖжөӢNATзұ»еһӢ (Full Cone, Restricted, Port Restricted, Symmetric)
  - ж”ҜжҢҒUDPе’ҢTCP
- **з«ҜеҸЈ**: 3478 (UDP), 3479 (TCP)

### 2. RelayжңҚеҠЎеҷЁ (TURN)
- **еҠҹиғҪ**:
  - еңЁP2PзӣҙиҝһеӨұиҙҘж—¶жҸҗдҫӣдёӯз»§
  - ж”ҜжҢҒUDP/TCPиҪ¬еҸ‘
  - еёҰе®Ҫз®ЎзҗҶе’ҢйҷҗжөҒ
- **з«ҜеҸЈ**: 50000-51000 (дёӯз»§з«ҜеҸЈжұ )

### 3. дҝЎд»ӨжңҚеҠЎеҷЁ
- **еҠҹиғҪ**:
  - и®ҫеӨҮжіЁеҶҢе’ҢеҸ‘зҺ°
  - SDPдәӨжҚўеҚҸи°ғ
  - иҝһжҺҘзҠ¶жҖҒз®ЎзҗҶ
  - Websocketй•ҝиҝһжҺҘ
- **з«ҜеҸЈ**: 8080 (WS), 8443 (WSS)

### 4. DIDжңҚеҠЎ
- **еҠҹиғҪ**:
  - и®ҫеӨҮиә«д»Ҫз”ҹжҲҗе’ҢйӘҢиҜҒ
  - еҹәдәҺе…¬й’Ҙзҡ„и®ҫеӨҮи®ӨиҜҒ
  - и®ҝй—®д»ӨзүҢз®ЎзҗҶ
- **з«ҜеҸЈ**: 9000 (HTTP)
