DCUtR protocol message

Corresponds to the HolePunch protobuf message:
```proto
message HolePunch {
  enum Type {
    CONNECT = 100;
    SYNC = 300;
  }
  required Type type = 1;
  repeated bytes ObsAddrs = 2;
}
```
