\doxysubsection{详细描述}
\begin{DoxyVerb}multistream-select 消息编码/解码模块

协议规范:
- 消息格式: varint长度前缀 + UTF-8字符串 + \n
- varint: 无符号变长整数编码 (multiformats unsigned varint)

参考: https://github.com/multiformats/unsigned-varint
\end{DoxyVerb}
