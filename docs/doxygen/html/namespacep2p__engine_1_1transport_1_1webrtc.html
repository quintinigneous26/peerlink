详细描述

WebRTC DataChannel 传输实现

基于 aiortc 库实现 WebRTC DataChannel 支持，
实现浏览器与 Python 之间的 P2P 通信。

协议ID: /webrtc-direct

特性:
- WebRTC DataChannel 支持
- ICE 候选交换
- DTLS-SRTP 加密
- 与浏览器 WebRTC 互操作
