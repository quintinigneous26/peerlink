详细描述

multistream-select 协议协商器

支持两种角色:
- Initiator: 发起协商，提议协议
- Responder: 响应协商，接受或拒绝协议

同时打开场景:
- 双方同时发送 multistream 协议 ID
- 后续通过 tie-breaker 决定主导权
