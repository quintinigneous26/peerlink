\begin{DoxyVerb}连接类型枚举

表示实际建立的连接类型，用于区分直连和中继连接。

属性:
    P2P_UDP: 直连UDP连接（最优）
    P2P_TCP: 直连TCP连接（备选）
    RELAY_UDP: 中继UDP连接（降级方案）
    RELAY_TCP: 中继TCP连接（降级方案）
    FAILED: 连接失败
