执行 TCP 打孔

原理：
TCP Simultaneous Open（RFC 793）
双方同时发起 TCP 连接，利用 SYN-SYN/ACK-ACK 三次握手

适用场景：
- 企业防火墙拦截 UDP
- UDP 限流严重
- 需要可靠传输
