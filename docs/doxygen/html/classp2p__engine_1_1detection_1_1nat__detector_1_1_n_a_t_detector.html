检测 NAT 类型

检测流程（RFC 3489 简化版）：
1. 向 Server A 发送请求 → 获取映射地址 M1
2. 向 Server B 发送请求 → 获取映射地址 M2
3. 比较 M1 和 M2：
   - 相同 → 非对称 NAT
   - 不同 → 对称 NAT
