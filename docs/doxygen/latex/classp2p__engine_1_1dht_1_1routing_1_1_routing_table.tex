\doxysubsection{详细描述}
\begin{DoxyVerb}Kademlia routing table with k-bucket tree structure.

Maintains peers in a binary tree of k-buckets based on XOR distance.
Each bucket covers a range of the ID space.
\end{DoxyVerb}
