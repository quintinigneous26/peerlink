\begin{DoxyVerb}决定是否需要降级

1. 双对称 NAT → 立即降级（0ms）
2. 移动 CGNAT + 对称 → 快速降级（<1s）
3. 跨运营商失败多次 → 降级
4. 跨境超时多次 → 降级
5. 单次打孔超时 → 重试，3次后降级
